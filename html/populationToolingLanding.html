BMMO NXE Toolbox (BMMO-XY) populationTooling

The BMMO-XY toolbox contains some tools to aid population studies, the documentation for the populationTooling is a set of MATLAB live scripts. These can be used to play around with the code examples when reading.
The set of live scripts regarding the populationTooling will be extended in the future, the ones currently available are:
  • demoPlots
  • bmmoPPT
  • bmmoStudy
The html version of the livescripts are also available in the BMMO-XY module documentation inside MATLAB.
