BMMO NXE Toolbox (BMMO-XY)

The BMMO NXE toolbox is a set of MATLAB tools for BMMO & BL3 NXE modelling and analysis.
The documentation for the toolbox consists of a set of MATLAB live scripts. These can be used to play around with the code examples when reading.
The set of live scripts will be extended in the future, the ones currently available are:
  • bmmo_help_getting_started
  • bmmo_help_apply_SBC
  • bmmo_help_inline_SDM
  • bmmo_help_KA_actuation
  • bmmo_help_rerunSimulation
  • populationTooling/demoPlots
  • populationTooling/demoPPT
  • populationTooling/demoStudy
